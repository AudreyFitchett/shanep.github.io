# Grading Rubric

Your grade will be calculated with the following weights and formula:

**Weights:**

- 70% - Testing
  - Number of tests Passed / Total Number of Tests
- 30% - General Coding Guidelines
  - Criteria detailed below

**Formula:**

$$Grade = (0.7 * Testing\%) + (0.3 * General\%)$$
