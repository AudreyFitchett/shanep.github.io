== What?

The process is one of the most fundamental abstractions that the OS provides to
users. A process is a running program.

* Address space
* Registers
** programming counter (PC)
** OR instruction pointer (IP)
* Stack pointer

== Process API

* Create
* Destroy
* Wait
* Miscellaneous Control
* Status
