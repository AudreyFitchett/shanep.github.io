== Fedora

NOTE: Updated to fedora 38

=== Install Tools

[,bash]
----
sudo dnf -y update`
sudo dnf -y groupinstall "Development Tools" "Development Libraries"
sudo dnf -y install asciidoctor
sudo dnf -y gnome-tweaks
----
